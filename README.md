# YouTube to Guides

A CLI tool that converts YouTube videos into clear, structured, step-by-step guides with images and text.

## Usage

```bash
ytguide generate <youtube-url>
```

## Installation

```bash
uv sync
```

## Development

```bash
# Lint
uv run ruff check .

# Format
uv run ruff format .

# Tests
uv run pytest
```
