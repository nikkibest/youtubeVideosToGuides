# Product Definition: YouTube to Guides

## Project Name

YouTube to Guides

## Description

A CLI tool for transforming YouTube video content into clear, structured, step-by-step guides.

## Problem Statement

Users watching tutorial videos (e.g. how to learn a handstand, fingerstyle guitar songs) have no easy way to capture the content as a structured, readable reference document. They need a tool that extracts the key steps, screenshots, and explanations from a video and assembles them into a well-organized guide they can use later — whether practising a physical skill or learning to play a piece of music.

**Examples:**
- A "How to learn handstand" video → a step-by-step guide with images and instructions for each stage of progression
- A fingerstyle guitar song on YouTube → a structured document with notes organized by section (intro, verse, chorus, bridge, outro) with clear notation per interval

## Target Users

- **Learners**: People who watch tutorial videos and want a structured reference document to consult during practice
- **Content consumers**: People who want offline, readable versions of YouTube guides for later use

## Key Goals

1. Convert any YouTube tutorial, training video, or instructional content into a clear, step-by-step document
2. Include relevant images and screenshots at each step for visual reference
3. Structure output intelligently based on video content type (e.g. skill progression, song sections)
